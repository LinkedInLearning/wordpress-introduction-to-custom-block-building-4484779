# WordPress: Introduction to Custom Block Building
This is the repository for the LinkedIn Learning course `WordPress: Introduction to Custom Block Building`. The full course is available from [LinkedIn Learning][lil-course-url].

![course-name-alt-text][lil-thumbnail-url] 

WordPress is a powerful tool for getting your content online, and its custom blocks can save you a lot of time. In this course, full stack engineer and WordPress expert Nyasha Green presents an introduction to custom block building in WordPress. Find out what a block is and how to customize it to make your WordPress projects easier and faster. Explore how to create your own custom blocks that you can use to show information more clearly on your sites, as well as how you can reuse custom blocks once they are created. This course equips you with the necessary skills to create, customize, and manage WordPress blocks, enhancing your ability to build dynamic, user-friendly websites and opening up new possibilities for content creation and presentation on the WordPress platform.
